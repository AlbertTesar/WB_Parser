# Wildberries Discount Tracker Bot

Course project developed at Kuban State University (KubSU).

A Telegram bot for monitoring prices and discounts on the Wildberries marketplace. Users can add links to products or catalog pages, and the bot periodically checks for price changes and sends notifications with summary reports.

## Features

- Monitoring of individual products and categories
- Automatic price tracking
- Discount change notifications
- Periodic summary reports
- Price history monitoring
- Telegram-based user interface

## Technologies

- Java
- Spring Boot
- Telegram Bot API
- PostgreSQL
- Jsoup
